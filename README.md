# Predictive Analytics Using Historical Data

## Overview
This project is an interactive Predictive Analytics Dashboard developed using Power BI to forecast future sales trends using historical business data from the Superstore dataset. The dashboard focuses on trend analysis, forecasting, and business intelligence visualization to help understand future sales performance and profitability patterns.

The project uses historical sales and profit data to identify growth trends, seasonal fluctuations, and category performance over time. Power BI forecasting features were used to generate future sales predictions along with confidence intervals for better analytical interpretation.

---

## Features

- KPI Cards for:
  - Total Sales
  - Average Profit
  - Total Profit
  - Total Orders

- Interactive Sales Forecast Analysis
- Historical Monthly Sales Trend
- Category Performance Ranking
- Profit Contribution Analysis
- Interactive Slicers for:
  - Region
  - Category
  - Segment

---

## Tools & Technologies Used

- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Excel Dataset
- Power BI Forecasting Analytics

## Dataset

The dashboard uses the Superstore Sales Dataset containing:
- Historical order data
- Sales and profit information
- Product categories
- Customer segments
- Regional business data
- Order dates for time-series forecasting

---

## Key Insights

- Historical sales trends indicate consistent business growth patterns.
- Forecast analysis predicts stable future sales performance.
- Seasonal fluctuations are visible in monthly sales trends.
- Technology category contributes significantly to overall sales growth.
- Profit contribution varies across different product categories.

---

## Dashboard Components

### Sales Forecast Analysis
Forecasts future sales trends using historical sales data and confidence intervals.

### Historical Monthly Sales Trend
Visualizes historical monthly sales performance over time.

### Category Performance Ranking
Analyzes how category rankings changed across different periods.

### Profit Contribution Analysis
Displays category-wise profit contribution using a waterfall chart.
